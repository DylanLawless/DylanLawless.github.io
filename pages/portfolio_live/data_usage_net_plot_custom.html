Data usage tracking networks

Long-term projects use data from a range of sources. Common databases or core files are often reused multiple times. Some data can remain dormant for long periods of time. We built automated data tracing systems that are used to present summarised analytics that are conducive to multipurpose usage and error prevention. This software scans entire data servers or project code to find and qunatify all data sources. It then produces live interactive reports without the need for manual curation.
